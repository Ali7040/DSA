# HackerRank Solutions

This directory contains solutions to HackerRank challenges organized by domain and category.

## Structure

```
hackerrank/
├── algorithms/        # Algorithm challenges
├── data-structures/   # Data structure problems
├── mathematics/       # Math-related problems
└── sql/              # SQL challenges
```

## Organization

- **Algorithms**: Organized by topic (warmup, implementation, strings, sorting, etc.)
- **Data Structures**: Organized by data structure type (arrays, linked-lists, trees, etc.)
- **SQL**: Organized by SQL topic (basic-select, advanced-select, aggregation, etc.)

Each category contains subdirectories for different programming languages:
- `cpp/` - C++ solutions
- `python/` - Python solutions
- `javascript/` - JavaScript/TypeScript solutions

## File Naming Convention

- Format: `challenge-name.ext`
- Example: `simple-array-sum.cpp`
- For multiple solutions: `simple-array-sum-approach1.cpp`, `simple-array-sum-approach2.cpp`
