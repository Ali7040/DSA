# LeetCode Solutions

This directory contains solutions to LeetCode problems organized by difficulty level and problem category.

## Structure

```
leetcode/
├── easy/          # Easy difficulty problems
├── medium/        # Medium difficulty problems
├── hard/          # Hard difficulty problems
└── sql/           # SQL problems (organized by difficulty)
```

## Organization

Each problem category folder contains subdirectories for different programming languages:
- `cpp/` - C++ solutions
- `python/` - Python solutions
- `javascript/` - JavaScript/TypeScript solutions

## File Naming Convention

- Format: `problem-number-problem-name.ext`
- Example: `0001-two-sum.cpp`
- For multiple solutions: `0001-two-sum-brute-force.cpp`, `0001-two-sum-optimized.cpp`

## Solution Format

Each solution file includes:
- Problem statement and link
- Multiple solution approaches
- Time and space complexity analysis
- Explanation of the optimal solution
- Test cases
