# Code Templates

This directory contains template files for different programming languages to help maintain consistency across solutions.

## Available Templates

- `cpp/template.cpp` - C++ template with standard structure
- `python/template.py` - Python template with type hints
- `javascript/template.js` - JavaScript template
- `sql/template.sql` - SQL template with multiple approaches

## Usage

Copy the appropriate template file when starting a new problem solution. This ensures:
- Consistent code structure
- Proper documentation format
- Complexity analysis sections
- Test case organization

## Template Features

Each template includes:
- Problem metadata (name, platform, difficulty, link)
- Multiple approach placeholders
- Complexity analysis sections
- Test case structure
- Comments explaining the optimal solution
